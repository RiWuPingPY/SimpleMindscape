# 脉络 · Mindscape

一个零依赖的**思维导图 / 逻辑（布尔）图**可视化工具。采用 Leaf Hub 的设计语言（暖米色背景、品红→紫罗兰渐变、玻璃拟态、星云氛围背景、Noto Serif SC 衬线字体）。

底层是**任意有向图**模型（节点 + 有向连线），既能画整齐的树状思维导图，也能画带环、共享、多对多关系的逻辑网络。

## 两种运行方式

### 1. 浏览器（最简单）
直接双击 `index.html` 用任意现代浏览器打开即可，无需安装。

### 2. 桌面 App（不依赖浏览器）
双击 **`StartDesktopVision.bat`**：
- 首次运行会自动 `npm install` 安装 Electron（需已装 [Node.js](https://nodejs.org)，仅第一次需联网）。
- 之后即以独立窗口启动，和浏览器无关。

打包成可分发的 exe（可选）：
```bash
npm install
npm run dist:portable   # 生成免安装绿色版到 release/
npm run dist            # 生成 NSIS 安装包 + 便携版
```

## 功能

- **双模式**
  - **思维导图**：自由结构，分层自动布局。
  - **逻辑图**：节点指定逻辑门（`AND / OR / NOT / XOR / VAR` 变量），实时求值、绿红着色，连线带方向箭头并同步染色。
- **任意图操作**
  - **凭空建节点**：双击空白处，或点工具栏「凭空建节点」。
  - **连线**：拖节点右侧端口到另一个节点即可连线；松手落在空白处会**新建节点并连接**。
  - **一连多 / 多连一 / 递归（成环、自环）**：连线不受树结构限制，随意连。
  - **删除连线**：点击连线选中后按 `Delete`，或点连线中点的 ✕。
  - **重连连线**：选中连线后拖动两端的圆形把手到别的节点。
- **节点尺寸自定义**：拖节点右下角手柄改宽高；检查器里「重置尺寸」恢复自适应。
- **排布方向**：横向 / 竖向 / **自适应**（按视口长宽比自动选方向）。连线按相对位置自动走左右或上下。
- **导出**：Markdown（带 `@AND/@VAR(1)` 注解，可反向导入；重复引用标 `(↗)`）、PNG（2× 高清、含箭头）、TXT、工程 JSON。
- **导入**：Markdown / 缩进文本 / 工程 JSON（含旧版 `parent` 工程自动兼容）。
- **文档历史 + 自动保存**：左侧栏像聊天记录一样列出所有文档，自动保存到浏览器本地（`localStorage`），随时切换/新建/删除，标题自动取自中心主题。
- **演示动画**：点「演示」，光点从每个独立母节点射出、沿连线流动并点亮沿途节点；可设循环（脉冲频率）/ 单次、速度与颜色。
- **颜色主题**：脉络 / 蔷薇 / 薄荷 / 青空 / 暖沙 / 暗夜，左侧栏底部切换，记忆偏好。
- **关于面板**：左侧栏底部「关于」，含作者信息（[pingyi.me](https://pingyi.me)）与项目链接。
- **多语言**：12 种界面语言 —— 简体中文 / 繁體中文 / English / 日本語 / 한국어 / Français / Deutsch / Español / Português / Русский / العربية（含 RTL）/ हिन्दी。首次按浏览器语言自动选择，可在左侧栏底部切换并记忆。

## 快捷键

| 键 | 作用 |
|----|------|
| 双击空白 | 凭空建节点 |
| 双击节点 | 编辑文本 |
| `Tab` | 给选中节点加子节点 |
| `Enter` | 凭空建节点 |
| `Delete` / `Backspace` | 删除选中的节点或连线 |
| `Esc` | 取消选中 |
| 滚轮 | 缩放 · 拖空白 | 平移 |

## Markdown 格式（可反向导入）

```markdown
# 中心主题

- 中心主题
  - 第一个分支 @AND
    - 条件 A @VAR(1)
    - 条件 B @VAR(0)
  - 第二个分支
    - 想法 B (↗)   ← 被多处引用的共享节点
```

> 注：跨层级的共享/成环连线在 Markdown 里以 `(↗)` 标记（不重复展开）。要**完整无损**保存图结构（含坐标、尺寸、所有连线），请用「保存工程 (.json)」。
