脉络 Mindscape

VERSION 1.0 · thought · logic atlas

一个零依赖、可离线运行的思维导图 / 逻辑（布尔）图可视化工具。底层是任意有向图模型，既能画整齐的树状导图，也能表达多对多、共享与递归成环的逻辑网络。

  • 思维导图 / 逻辑图双模式，逻辑门实时求值着色
  • 凭空建节点、自由连线、删除 / 重连、节点尺寸自定义
  • 横向 / 竖向 / 自适应自动布局
  • 沿连线流动的光点演示（脉冲循环 / 单次）
  • 聊天式文档历史，自动保存 / 加载
  • 多套颜色主题；导出 Markdown / PNG / TXT / 工程 JSON
技术：原生 HTML/CSS/JavaScript（零运行时依赖）+ 可选 Electron 桌面封装。
设计语言取自 Leaf Hub。许可证 MIT。
作者：一位独立全栈开发者（Leaf Hub）。
